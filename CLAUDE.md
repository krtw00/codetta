- Plane project: `codenica/CDT` — https://plane.codenica.dev/codenica/projects/1f3f9abc-63f7-4a28-9781-38e8eb3e0368/issues/

- 状況確認は Web UI、 または Claude code セッション内の plane MCP tool (`mcp__plane__list_work_items` 等)
