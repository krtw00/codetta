# Codetta

> LLM ネイティブな軽量音楽制作ツール / **The DAW for LLMs**

Codetta は Claude / ChatGPT 等の LLM が **読み・書き・編集** できる音楽プロジェクト形式と、
MCP (Model Context Protocol) 統合により、AI が一級市民として作曲に参加できる
最初の OSS 音楽制作環境を目指します。
